## 👵 Elderly Drive: 고령 운전자 사고 분석 및 정책 제안

### 1. 개요
본 프로젝트는 숙명여자대학교 경영정보처리론 01분반 기말 과제로써, 고령 운전자 교통사고 데이터를 분석하여 위험성을 진단하고, 면허 반납 정책의 실효성을 높이기 위한 이동권 보장 방안을 제언합니다.

### 2. 주요 분석 내용
- **연령별 사고 위험성 진단:** 65세 이상 고령층의 상대 위험도 분석
- **지역별 면허 반납 현황:** 지자체별 반납율 시각화 및 정책 시뮬레이션
- **이동권과 반납율의 상관관계:** 대중교통 인프라가 자발적 반납에 미치는 영향 분석
